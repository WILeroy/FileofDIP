# UCAS Course Code

国科大一些课程的相关代码, 以下是详细列表:

* 图像处理与分析(19-20秋季学期)

  * utils.py, 功能函数, 包括二值化, RGB转灰度图像, 图像边界填充等.

  * deblur.py, 盲去运动模糊.
  * enhancement.py, 几种图像增强的方法.
  * morphology.py, 形态学基本操作与骨架提取算法.
  * wtfilter.py, 小波域维纳滤波.

* 算法中的最优化方法(19-20秋季学期)

  * datapre.py, 2017美赛D题数据预处理与建模.

* 矩阵分析与应用(19-20秋季学期)

  * decomposition.py, 矩阵分解, 包括LU分解, Givens分解, Household分解等.
